## ĐẶC ĐIỂM THIẾT KẾ

### Style & Theme
- **Thiết kế**: Đơn giản, gần gũi, phản ánh văn hóa Mường
- **Màu sắc**: Có thể sử dụng tông màu tự nhiên, gỗ, đất (phản ánh văn hóa truyền thống)
- **Yếu tố văn hóa**: 
  - Hàng rào
  - Đường đi
  - Sắp đặt đồ đạc
  - Bàn thờ thổ công
  - Nhà sàn Mường

### Layout Structure
- **Header**: Fixed hoặc sticky navigation
- **Main Content**: Grid hoặc list layout cho tin tức
- **Sidebar**: Thông tin tham quan, dịch vụ
- **Footer**: Multi-column với thông tin liên hệ, links, social media

### Responsive Design
- Website cần responsive cho mobile và tablet
- Navigation có thể collapse thành hamburger menu trên mobile

## CÁC TRANG CHÍNH
